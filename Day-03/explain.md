## What is Linux?
Linux is an operating system like Windows or macOS.
It is open-source, which means anyone can view and modify its code.

## Why is Linux Important for Ethical Hacking?
Most cybersecurity and ethical hacking tools are designed to run on Linux.
Linux provides better control, stability, and security for these tasks.

## Linux File System (Basic Idea)
Linux uses a tree-like structure.
All files and folders start from a root directory.

Folders such as home, bin, and etc store different types of files and system data.

## What are Linux Commands?
Linux commands are used to interact with the system using a terminal.
Commands help users navigate folders, manage files, and run tools.

## Why Students Should Learn Linux
Learning Linux makes it easier to understand how systems work.
It is a core requirement for anyone entering cybersecurity or ethical hacking.
