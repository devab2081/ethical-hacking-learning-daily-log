##  What I Learned Today  

Today I learned the basics of Linux.  
I understood that Linux is an operating system that is widely used in cybersecurity and ethical hacking.

I learned that Linux uses a different file system structure compared to Windows.  
Everything in Linux is treated as a file, including devices and folders.

I practiced some basic commands like checking the current directory, listing files, and creating folders.  
I realized that knowing Linux commands is very important because most security tools run on Linux.

Linux is preferred in ethical hacking because it gives more control over the system and is open-source.

##  Key Takeaways

- Command-line usage is a basic skill
