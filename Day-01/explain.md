## What is Ethical Hacking?
Ethical hacking means testing a system or application with permission to find security weaknesses.

## Why is it Needed?
Many websites store personal and sensitive data.
If security is weak, attackers can misuse that data.
Ethical hackers help find problems before attackers do.

## Is Ethical Hacking Legal?
Yes, only when permission is given.
Without permission, hacking is illegal.

## Who Should Learn This?
- Students interested in cybersecurity
- Developers who want to secure applications
- Anyone curious about how attacks happen
